Privacy Policy

Last updated: March 4, 2026

Overview

HFO Calendar Integration is a private utility application. It is not available to the general public and is used exclusively by its owner for personal calendar management.

Data Collection

This application accesses Google Calendar data solely to read and create calendar events for the authorized user. No data is collected, stored on external servers, shared with third parties, or used for any purpose other than displaying and managing the user's own calendar.

Data Storage

OAuth tokens are stored locally on the user's own machine. No user data is transmitted to or stored on any external server.

Contact

For questions about this policy, contact the application owner directly.
